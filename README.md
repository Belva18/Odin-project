**Belva's Tech Studio** is an assignment from odin project designed to showcase and graphic design. This project is part of a dedicated series of 10 IT-related projects being developed during my second year at **Wisconsin International University College**.
##  Project Overview
This application should serves as a digital hub for clients to explore technical services and get in touch with the studio. It focuses on clean UI/UX principles and responsive design.
### Core Services
 * **Web Development:** Building modern, responsive websites.
 * **System Applications:** Developing functional backend and desktop solutions.
 * **Logo and Flyer Design:** Professional graphic design and branding services.
## 🛠️ Built With
 * **HTML5:** Structured the content and navigation.
 * **CSS3:** Implemented Flexbox layouts, custom box styling, and responsive padding.
 * **VS CODE** Used for initial UI/UX prototyping and design concepts.
## Features
 * **Navigation Bar:** Easy access to "About" and "Address" sections.
 * **Service Showcase:** A visual grid highlighting core studio offerings.
 * **Call to Action:** A dedicated section for user engagement and sign-ups.
